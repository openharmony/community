# SIG_Security
简体中文 | [English](./sig_security.md)

说明：本SIG的内容遵循OpenHarmony的PMC管理章程 [README](/zh/pmc.md)中描述的约定。

## SIG组工作目标和范围

### 工作目标
OpenHarmony安全业务的开发规划

### 工作范围
OpenHarmony安全业务,包括应用权限、应用签名、设备认证和密钥管理等
